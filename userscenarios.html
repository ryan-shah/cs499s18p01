User Scenarios

  • Logging in

    1. User goes to the Spydr login page
    2. User will log in and be redirected to the management console

  • User adds a new program to the database

    1. Starting at the managment console, the user will select the "Add Program" button
    2. This will direct them to the "Add Program" form
    3. The User will input the following information about the program:
      • Name
      • Absolute path
      • Estimated Memory Usage
      • Dependencies
    4. The User will press "Submit" or "Cancel" to preform that action on the database
    5. The User will be redirected to the management console

  • User edits an existing program

    1. Starting at the management console, the user will select a program from the list on the left to edit
    2. The User will then be redirected to the information page of that program
    3. The User can then press the "Edit" button and change any information they wish
    4. Once done, the User will press the "Submit", "Delete", or "Cancel" button to make the corresponding changes to the database
    5. The User will be redirected to the management console

  • Add a program to the list of programs to execute

    1. Starting at the management console, the user will select a program not already in the run list on the list on the left
    2. The User will then be redirected to the information page for that program
    3. The User will press the corresponding "Add to List" button
    4. The User will be redirected to the managment console

  • Remove a program from the list of programs to execute

    1. Starting at the management console, the user will select a program already in the run list on the list on the left
    2. The User will then be redirected to the information page for that program
    3. The User will press the corresponding "Remove from List" button
    4. The User will be redirected to the managment console

  • Change the time programs are run

    1. Starting from the management console the User will select the "Change Time" button on the top navigation bar
    2. The User will select a new time from resulting popup window
    3. Once done, the User will press the "Change" or "Cancel" button to make the corresponding change
    4. The User will be redirected to the management console
