User Scenarios

  • Logging in
    1. User goes to the Spydr homepage
    2. User will log in and be redirected to the management console
  • User adds a new program to the database
    1. Starting at the managment console, the user will select the "Add Program" button
    2. This will direct them to the "Add Program" form
    4. The User will input the following information about the program:
      • Name
      • Absolute path
      • Estimated Memory Usage
      • Dependencies
    5. The user will press "Submit" or "Cancel" to preform that action on the database
    6. The user will be redirected to the management console
  • Editing an existing program
